Phones hide individual lenses (main/ultra-wide/tele) behind one "Back Camera" — the OS switches lenses based on this zoom value, not a separate camera picker. Try values below 1.0 for ultra-wide, above ~2 for telephoto (varies by phone).
Rejects a scan if it fails its checksum (catches garbled digits, e.g. from motion blur), and requires the same code twice in a row before accepting. Adds a tiny delay but fixes occasional wrong-digit misreads. Turn off only if scanning feels too slow.
Tip: change one setting at a time, rescan the same barcode a few times, and watch ms/read and scans/sec to compare.
